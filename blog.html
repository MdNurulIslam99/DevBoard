Question-1: What are the different ways to select an element in the DOM?

There are multiple ways to select an element in the DOM in JavaScript.This multiple way are getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

These three properties are used to manipulate the content of an HTML element, but they work differently.InnerHtml work sets the HTML content inside an element dynamically and InnerText work only visible text inside an element also ignores hidden elements.

Question-3: What is event delegation in the DOM?

Event Delegation handles events in JavaScript.It works parent element handles events for its child elements and added dynamically.

Question-4: What is event bubbling in the DOM?

Event Bubbling is when an event happens on an element and it automatically moves up to its parent elements.
